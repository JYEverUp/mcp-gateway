MCP 调试页面导航

原来的调试页面内容太长，我已经按职责拆成几个独立页面。你可以从这里进入不同的功能页，分别做连接调试、网关管理、工具管理和 AI 联调。

连接调试

建立会话、健康检查、发送 initialize/tools/list/resources/list/tools/call。

网关管理

处理网关注册、详情查询、列表查询和鉴权更新。

工具管理

处理工具注册、工具列表和协议更新，默认对齐当前 SQL 工具配置。

AI 聊天

拉取工具并调用 chat-with-tools，单独验证模型联调链路。

旧版整合页

保留原始长页面作为兼容入口，后续你要的话我可以继续把它也收口成这个导航页。
